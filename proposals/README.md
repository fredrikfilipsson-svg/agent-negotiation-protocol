# ANP proposals

Draft RFCs for evolving the protocol beyond 0.1. Nothing in this
directory is normative; `protocol/SPEC.md` remains the only
specification. Each proposal is discussed in a GitHub issue and, where
practical, prototyped behind the `ANP_DRAFT=1` flag of
`scripts/mock-buyer-host.mjs` so the discussion has running code.

Rendered at <https://aiaagentnetwork.com/proposals>.
